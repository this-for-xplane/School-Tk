## 주요 기능
- 시간표 작성 및 관리
- 캘린더 메모 기능
- 다크모드 지원
- 다국어 지원 (한국어, 영어, 일본어, 중국어)
- 모바일/태블릿 반응형 디자인

## 기술 스택
- HTML5
- CSS3
- Vanilla JavaScript
